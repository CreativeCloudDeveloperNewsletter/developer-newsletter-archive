  ExtendScript Debugger for VSCode is Here for Mac  
  Our VSCode plugin for debugging ExtendScript for macOS is out now in the VSCode Marketplace, go get it.

Please note that this is not a Windows release. Even though you can install v.1.0.0 on Windows, it is currently in beta. A Windows release should be out in matter of weeks, but for now the most stable Windows release is v0.2.4 in our prerelease (login required), in the downloads section.

Follow the Adobe Tech Blog for more updates on the ESTK Debugger.

  Update your manifest!  
  We've updated the I/O Developer Console to improve the XD plugin submission and review process. Plugin developers who are going to update their current plugins or listings will need to update their manifests before submitting updates to the I/O Console.

Read more about changes to the I/O Console workflow on our blog.

  XD 17 Changes the Edit Context  
  Last week’s release of XD 17 loosened the “edit context” rules concerning which nodes your plugin may edit or select. Plugins now have the ability to edit many (but still not all!) of the nodes in the subtree containing the selected nodes.…
