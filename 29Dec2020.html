Reminder: Adobe’s Winter Shutdown
Our integrations review staff for the Exchange Marketplace, the XD Plugin Manager, and service APIs will be celebrating the holidays and offline from December 24th, 2020 through January 3rd, 2021. We will resume reviews from January 3rd, 2021. Any submissions not finalized before the break or received during the break will be handled as quickly as possible from January 3rd onward.
ARM Processor Update

As of Creative Cloud Desktop App Version 5.3.5.518, plugin and extension installation is not supported for users of ARM processors, (with the exception of the Adobe XD Plugin Manager). We regret this inconvenience and expect a fix in the next version of the Creative Cloud Desktop App due at the end of February 2021. Unfortunately, at the time of this writing, alternate installation methods like ExManCmd and third party tools like Anastasiy's Extension Manager cannot install plugins on ARM devices. It is not possible to roll back the Creative Cloud Desktop App. We will update you as soon as a workaround is available.

For more information on ARM support for CEP and UXP based plugins, please see this article on the Adobe Tech Blog.

InDesign Server Dec 2020 is Shipping Now
InDesign Server 2020/v16.0 is shipping now. Read the Release Notes here. Licensing is now based on the use of the Creative Cloud Admin Console to create an installation package rather than the use of serial numbers, see our blog post from earlier this year for further details.

Adobe will not be offering a watermark version of the product going forward. Commercial product developers who develop products or solutions for clients can request a license through their membership in the Adobe Exchange Partner Program for Creative Cloud. In-house developers should purchase the Development license.

XD 36 Contains Big Updates for Plugin Developers

XD 36 features support for Spectrum UXP, a new cross-app manifest, and support for the new UXP Developer Tool (first supported on Adobe Photoshop 2021). XD 36 also brings some new layout APIs.

Read more about it on the Adobe Tech Blog, or in the prerelease.
