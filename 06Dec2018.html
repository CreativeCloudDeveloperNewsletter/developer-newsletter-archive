Adobe Exchange Program News
On December 14th, the Adobe Exchange Program will have two new features:
  1. Easy signup – We’ve greatly simplified the signup process for the Adobe Exchange Program. All you need is your Adobe ID to get started, and the program is now totally free!
  2. Create Trials – A long requested feature is the ability to create trials. We now provide this feature for any paid perpetual product listing on the Adobe Exchange marketplace.
Look out for these exciting new features from December 14.
