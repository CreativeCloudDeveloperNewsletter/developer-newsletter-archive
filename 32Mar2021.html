Recompile Your C++ Plugins
  • Photoshop C++ plugins need to be recompiled to work on Apple silicon devices when running Photoshop natively.
  • Illustrator developers need to recompile in order to update the PiPL file to appear in Illustrator v25.1+.
  • InDesign developers of C++ plugins will have access to the 16.3 SDK in the InDesign Developer prerelease group soon. Read this blog post for details; a recompile is mandatory to keep InDesign plugins running on InDesign v16.3+ on macOS.

ExManCmd 7.11.1 Update & FAQ
ExManCmd 7.11.1 was released in March. This update works on computers using Apple silicon processors. The Creative Cloud Desktop app dependency from 7.11 has been rolled back. Read this new article for an ExManCmd FAQ, including the new log paths and version compatibility information.
Read the FAQ
CEP 11 is Coming This Summer
CEP 11 is coming to prerelease for every Creative Cloud app other than Photoshop this summer. CEP 11 includes Apple silicon support, and a CEF update to version v88. This is a major update from CEP 10 (CEF v74). Be sure to join a prerelease for the app(s) you support and test your product against CEP 11.
