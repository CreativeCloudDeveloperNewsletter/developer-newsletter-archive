InDesign 18.5 Supports UXP Plugins
Now you can build UXP plugins for InDesign! Get started with the following:
  • Read the docs to write a simple plugin, understand crucial concepts of UXP, and find recipes for common use cases. Please note that some of your previously bookmarked URLs may be broken since the docs were restructured.
  • Check out this repo to find new samples and templates (more to follow soon).
  • Participate in the discussion in the new InDesign forum category for UXP developers.
  • Join the InDesign Office Hours.

UXP Developer Tools Update
UXP Developer Tools v1.9.0 is now available to all users in the Creative Cloud desktop app.

UXP & InDesign:
  • Support for UXP plugins with InDesign 18.5 and higher.
  • Arguments support for debugging UXP scripts with InDesign 18.5 and higher.
  • New starter templates to assist you with plugin creation.
Enhancements:
  • "Create Plugin" now automatically creates a new folder before adding all files to it.
  • UDT will now remember the main window and debug window size.
  • Node 17 support for the React starter plugin.
  • Fixed shortcuts, and other minor enhancements to templates.
