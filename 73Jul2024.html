Important InDesign C++ SDK Updates

InDesign version 20.0, planned for October 2024, has the following important updates that are likely to impact your plug-ins and build workflows:

  • Windows Compiler Upgrade to version 17.9.6. - The team recommends moving to Visual Studio 2022 Compiler v17.9.6 MSVC v143 with Windows 11 SDK 10.0.22621.0.
  • Xcode MacOS Compiler Upgrade to Xcode 15.2, supported on macOS 13.5 or later
  • C++ 17 will be the minimum required C++ version
  • An update to the flag _ITERATOR_DEBUG_LEVEL may affect Windows users.

Please read the complete update in the InDesign Developers Prerelease forums.
