InDesign Developer Prerelease Update

InDesign C++ Plugin Developers can now access the Debug version of InDesign 2021/v16.0 through the Creative Cloud Desktop App. First, please enroll in the InDesign Developers Prerelease program: prerelease.adobe.com. There will be a delay of a week or so before your Adobe ID is added to the distribution for Debug. That version will show up in the Creative Cloud Desktop Apps under the left panel, Prereleases.

Please note: If you’re already in the InDesign Developer prerelease program, you may need to sign in to accept the new terms before you see InDesign in the Creative Cloud Desktop App. If you still don’t see the prerelease, sign out of the Creative Cloud Desktop App and sign back in.

The latest InDesign SDK and related developer tools can be found at adobe.io.

Join Our LinkedIn Group for Developers
