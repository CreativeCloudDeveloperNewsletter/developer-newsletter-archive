UXP Type Definitions
Type definitions for UXP for v7.3 is now available on public npm. You can install the UXP types using npm install @adobe/cc-ext-uxp-types. These are also packaged with the UDT Playground in UDT v2.0.

Mini editors in Adobe Express Embed SDK v4
The upcoming v4.0 release of the Embed SDK allows developers to embed contextually relevant Express capabilities using our new mini editors. With mini editors, users can quickly and easily edit images while leveraging powerful AI-powered features such as Generative Fill and Remove Background. Join our community to stay up-to-date!
