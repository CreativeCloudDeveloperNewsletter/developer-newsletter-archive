UXP Updates: Scripting, Webview, C++
UXP 6.0 will be released with Photoshop 23.3.0, expected April 11th, (currently in prerelease). UXP 6.0 should not require any changes to your existing plugins, and it has exciting new features. Please note: You must update your manifest file to take advantage of the new features.
  • Webview support: UXP 6.0.2 supports webview for modal dialogs only, and only for remote content. Load any secure website, including your own backend if you have one.
  • Launch apps: In UXP 6.0.2 manifest v5 enables the ability to launch apps using their URI scheme.
  • Scripting: UXP 6.1 (in progress), unlocks scripting and allows writing UXP code without having to specify a full manifest.
  • C++: UXP 6.1 will also allow you to bundle a Photoshop C++ plug-in filter or native C++ binary in your UXP plugin and distribute it via the Creative Cloud Desktop application.
UXP 6.0 and 6.1 will also come with numerous bug fixes and other improvements.

Install UXP Plugins from the Command Line
Starting this month with Creative Cloud desktop app 5.7 (rolling out from April 6th to the 19th), you can install UXP plugins from the command line. Please note that unlike CEP installation workflows, you must use the Unified Plugin Installer Agent (UPIA) to ensure the host app recognizes your plugin.
