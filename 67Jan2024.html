Important InDesign C++ File Path Update
In August 2023 we announced POSIX style paths would replace HFS as the default in InDesign 19.x. As a follow up, please note: 
  • For InDesign 19.2, by default we will support HFS Path Styles but developers can enable POSIX Support through Cookies.
  • For InDesign 19.3, we will enable POSIX Path Styles by default.
(For InDesign SDK prerelease access, email wwds@adobe.com)
