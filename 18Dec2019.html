  Adobe’s Winter Shutdown  
 

Adobe offices will shut down from December 23rd through January 1st.

Our integrations review staff for the Exchange and XD Plugin Manager will be celebrating the holidays and offline from December 24, 2019 through January 1, 2020. We will resume reviews from January 2, 2020. Any submissions not finalized before the break or received during the break will be handled as quickly as possible from January 2.

We’ll see you in the new year! Take a look back at all we’ve done in 2019 in this year-in-review blog post.

  InDesign Server: New version shipping along with Pricing Changes  
 

The latest version of InDesign Server (v15.0) shipped on Dec 17th. Prices for InDesign Server increased approximately 10% in December as well.

One note for Mac users: The new version does not yet support Mac OS 10.15 (Catalina), only the older versions. Catalina support is expected to be in place in March 2020. See full technical specs here.

Send an email to sharewithids@adobe.com if you need to test on Catalina sooner than March.
