ExManCmd To Be Replaced
The command line plugin installer, ExManCmd, will be replaced by a new tool, the Unified Plugin Installer (UPIA), beginning in June with the rollout of Creative Cloud Desktop app 5.5. UPIA will be automatically installed with Creative Cloud Desktop 5.5, so no action will be necessary for most users. Users who invoke ExManCmd in by running ./ExManCmd --install, particularly in automated processes, will need to update their commands to ./UPIA --install.
