XD 21 API News: Panels & UXP 3.1
XD 21.1 is out now, and it includes a big under-the-hood transition to UXP 3.1. We recommend testing your plugins against XD 21.1, and reading through our list of known bugs and UI issues.

With XD 21.1, panel UI APIs have arrived! Although you can technically go live with a panel version of your plugin now, but Adobe will not be doing any marketing push around panels or partners that use panel APIs until August.

ExtendScript Debugger 1.1.2 is Out Now
VSCode 1.36 caused issues with the ExtendScript Debugger plugin for Windows, but after less than a week, we were able to release a fix with version 1.1.2.

Help Improve ExtendScript Debugger
If you’re just getting started with the debugger, share your crash logs with us and help us improve the experience. Here’s how:

Add "trace": true to your launch configuration file. Share the logs that appear in the Debug Console, either in our Developer Prerelease forum or in our public Adobe forums.

Let us know which mode you’re using the extension (the extensionMode configuration as described in the marketplace). The default is active, but you might also try passive mode for better stability.
