CEP 12 Available in Photoshop Beta

CEP 12 is currently available in Photoshop Beta version 25.12 (Intel version only), via the Creative Cloud Desktop app. CEP 12 includes a Chromium update to CEF v99, a migration from OpenGL to Metal in Mac OS, and Node version v17.7.1. An early version of the CEP 12 cookbook is located on a github branch. Please test your CEP plugins and leave feedback in the forum thread with the button below.
