New Developer Distribution Portal for Listing UXP Plugins in the Creative Cloud Marketplace
If you have a UXP plugin listed in our Marketplace, you can now add a video to the listing to introduce your plugin. To do so, you need to submit a new version of the plugin through the new developer distribution portal! This update is for developers listing UXP plugins only - CEP support is coming later this year. Creative Cloud Libraries API News
Please note that as of March 28th, 2023 we are no longer accepting new Creative Cloud Libraries integrations. We are taking some time to revamp the API, incorporating feedback and new features. Existing third-party integrations can continue being managed via the Adobe Developer Console and will still be able to reference the existing documentation.
