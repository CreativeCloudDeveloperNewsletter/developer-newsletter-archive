UXP v6.3 Updates
UXP v6.3 is integrated in Photoshop v24.0, InDesign v18.0 and XD v55.0. This version adds support for Blob and Crypto APIs, as well as new updates to the File System and Plugin Manager APIs.

UXP Developer Tool Updates
UXP Developer Tool v1.7.0 (UDT) can now debug UXP scripts in both InDesign and Photoshop. UDT v1.7 includes new templates and WebView samples, as well as enhancements and bug fixes.
