XD News; New API features, Open Source Resources
Since XD introduced third-party plugins last fall, the plugin APIs continue to evolve.

Did you miss December’s release of XD? You can read all the details on the Adobe Tech Blog. In the coming weeks we will publish an article about the upcoming January release of XD 15, which will include more updates for plugin developers.

Love open source? Explore our XD Awesome repo on GitHub. We would love to see your open source plugins linked from our repo; submit to XD Awesome.

If you’re excited about XD plugins, and have feedback, suggestions, plugin requests, or you’re considering building your first XD plugin, be sure to join our forums and chat with developers as well as members of the XD team.

InDesign Server News
We have recently updated the InDesign Server partner listings. There is a partner page under the InDesign Server product page. Currently this is only showing on the US site for adobe.com but we are localizing for other regions in the next few weeks. Check out the listings here. 

The InDesign Server partner page highlights technology partners offering workflow systems, automation, web to print and more. Because these solutions go beyond plugins, we have added a filter for these partners in the Partner Finder.

Over the next few months, we will also add sections for VARs, System Integrators, Trainers, Consultants and implementation partners that don’t sell a product but can work with customers.

If you want to get listed as a technology partner or a service provider, email the help desk at CCExchange@adobe.com.
