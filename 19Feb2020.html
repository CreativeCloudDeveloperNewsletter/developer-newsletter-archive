  Premiere Pro: New World Scripting Progress Report  
 

The “New World” scripting environment in Premiere Pro provides a modern JavaScript engine for ExtendScript while aiming to support existing ExtendScript code, without modification. “New World” scripting has been available on an opt-in basis since Premiere Pro 13.1. 
 
The next release of Premiere Pro and Adobe Media Encoder will default to "New World" ExtendScript environments with “Old World” behavior remaining accessible as-needed. Find additional details in the link below.
